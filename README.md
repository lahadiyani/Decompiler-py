# Encrypt & Decrypt Base64 and Marshall

Program ini adalah sebuah skrip Python sederhana yang dapat melakukan enkripsi dan dekripsi menggunakan teknik Base64 dan Marshall. Program ini dibuat untuk tujuan pembelajaran dan demonstrasi. Gunakan dengan bijaksana dan bertanggung jawab.

## Cara Penggunaan

1. Pastikan Anda telah menginstal Python di perangkat Anda.

2. Clone repo ini atau unduh kode program dari GitHub.

3. Jalankan program menggunakan Python di terminal atau command prompt:

```
python nama_program.py
```

4. Pilih opsi untuk melakukan enkripsi atau dekripsi.

## Opsi Menu

### Menu Enkripsi

- Encrypt Base16
- Encrypt Base32
- Encrypt Base64
- Encrypt Hex
- Encrypt Marshal
- Compile py > pyc
- Encrypt Marshal Zlib Base64
- Encrypt Zlib

### Menu Dekripsi

- Decrypt base64.b16decode
- Decrypt base64.b32decode
- Decrypt base64.b64decode
- Decrypt Hex
- Decrypt Marshal
- Uncompyle6 pyc > py
- Decrypt Marshal, Zlib, Base64
- Decrypt Zlib
